# Jogja After Dark — Clean No Numpuk

Versi ini dibuat untuk mengganti halaman lama yang panelnya saling menumpuk.

## Isi
- `index.html` saja
- CSS, JavaScript, dan data sudah digabung dalam satu file
- Layout tidak memakai panel floating/fixed
- Peta dan kontrol dipisah dalam grid kiri-kanan

## Cara upload
1. Hapus isi repository lama agar tidak bentrok.
2. Upload `index.html` dari ZIP ini ke root repository.
3. Aktifkan GitHub Pages:
   Settings → Pages → Deploy from branch → main → /root.
