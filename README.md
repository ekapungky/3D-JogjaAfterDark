# Jogja After Dark — GitHub Ready

Versi ini sengaja dibuat **satu file HTML** supaya upload ke GitHub tidak rusak dan tidak ada masalah folder CSS/JS/assets.

## Cara upload
1. Extract ZIP.
2. Upload `index.html` ke repository GitHub.
3. Buka Settings → Pages.
4. Source: Deploy from branch.
5. Branch: main, folder: /root.
6. Save.

File sudah berisi tabel lokasi, target output, WebGIS Leaflet, story map, simulasi 3D, survival mode, infografis, dan arahan laporan.
