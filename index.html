Model 3D Kota

Simulasi visual 3D sederhana: tinggi kolom mewakili haunted index tiap zona. Untuk final bisa diganti model GLTF/Three.js.

Mini Game / Survival Mode

Demo ringan: pengguna mengumpulkan 5 relik dari 5 zona sebagai elemen gamifikasi WebGIS.

🎮 Relik Kutukan

🧭 Aturan Survival Mode

  • Mulai dari Jalan Malioboro – Alun-Alun.
  • Lanjut ke Keraton dan Taman Sari sebagai area inti budaya-historis.
  • Ekstensi rute menuju Kota Gede dan Gua Selarong.
  • Zona dengan haunted index tinggi diberi prioritas efek visual dan storytelling lebih intens.
